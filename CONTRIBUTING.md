# 欢迎参与 Conflux

感谢你对这个项目感兴趣。无论你是想提建议、报 bug、还是直接写代码，这里都欢迎你。

## 怎么参与

### 💡 提建议 / 报 Bug

如果你有一个想法，或者发现了什么问题，可以直接开一个 [Issue](https://github.com/oyyye666-dev/Conflux/issues/new)，描述清楚就好，不需要用固定的模板。

### 🧑‍💻 提交代码

1. Fork 这个仓库
2. 在你自己的分支上做改动
3. 提交 Pull Request（PR），简单描述一下你改了什么、为什么改

如果你的 PR 还在构思中，可以在标题前面加 `[WIP]`，这样大家就知道还在进行中。

## 沟通风格

保持开放和尊重。技术可以争论，人不应该被冒犯。如果你觉得某句话不舒服，直接说出来。

## 项目方向

现在这个项目还没有固定的方向——这正是它有意思的地方。如果你有想做的事情，提出来大家一起讨论，方向本身就是需要被共创的东西。
